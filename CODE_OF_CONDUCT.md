# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. By participating in this community — issues, pull requests,
discussions, or any other space connected to the project — you agree to
uphold it.

The full text is available here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report unacceptable behavior, please use GitHub's private vulnerability
reporting flow as a confidential channel:

👉 <https://github.com/Thea-Labs/resume-cli/security/advisories/new>

(The same channel doubles for security reports — see
[`SECURITY.md`](SECURITY.md). It's the only private channel this project
guarantees.)

All reports will be reviewed and investigated promptly and fairly.
Maintainers are obligated to respect the privacy of the reporter.

## Enforcement

Maintainers will follow the Contributor Covenant's
[Community Impact Guidelines][guidelines] when determining the consequences
of any action they deem in violation.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
